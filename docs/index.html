JBox API documentation

Basics

init(config)

Creates a JBox game.

  • config - configuration object used by JBox to create a game loop. The accepted fields are:
    • init(null) - function that runs once at the beginning of the game
    • draw(null) - function that runs every frame before the screen buffer is drawn to the canvas
    • update(dt) - function that runs every frame after the screen buffer is drawn to the canvas. The dt parameter is the amount of time (in seconds) that elapsed since the last frame

Graphics

pget(x, y)
pset(x, y, [c])

Returns or sets the (c)olor of the pixel at coordinates x and y. If you don't specify color, the current draw color is used.

sget(x, y)
sset(x, y, [c])

Returns or sets the (c)olor of the pixel at coordinates x and y of the spritesheet. If you don't specify color, the current draw color is used.

fget(sprite, flagNumber)
fset(sprite, flagNumber, [c])

Returns or sets the (c)olor of the pixel at coordinates x and y of the spritesheet. If you don't specify color, the current draw color is used.
